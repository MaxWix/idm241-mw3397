Triggers:

hover on featured story thumnail, hover off, on click

Rules:

On hover, the title and opacity box on the bottom slide up enough for a description to slide up from the bottom. the whole box enlarges slightly. On click, the hyperlinked page is accessed as well as all data and code needed for that page, and the new page loads.
